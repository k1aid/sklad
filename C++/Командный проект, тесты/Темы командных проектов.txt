1. Реализовать игру "Морской бой!.
	Нужно реализовать два режима игры: человек-человек, человек-компьютер.
	Программа должна позволять пользователю выполнить вручную расстановку своих кораблей.
- Lysenko
- Lobozev
2. Реализовать игру "Викторина".
	Пользователь выбирает раздел знаний (например: игры, кино, музыка, путешествия и т.д.), и получает набор вопросов по выбранной тематике.
	У каждого вопроса должно быть четыре варианта ответа.
	У каждого вопроса есть свой коэффициент сложности.
	За каждый правильный ответ пользователь получает один балл, умноженный на коэффициент сложности.
	Когда пользователь ответит на все вопросы, на экране отображается набранная сумма баллов.
	Для повышения уровня сложности приложения информацию можно читать и записывать в файл.
	Также для повышения уровня сложности можно реализовать механизм турнирной таблицы (отображает позицию пользователя по конкретному разделу знаний).
- Газарян
- Дымочкина
3. Реализовать приложение для подсчёта калорий.
	Пользователь вводит информацию об употреблённых продуктах, приложение выдаёт ему количество полученных калорий.
	Обязательно реализовать механизм регистрации пользователя.
	При регистрации пользователь заполняет ник, пароль, рост и вес.
- Бочаров
- Трещеткин
- Середа
